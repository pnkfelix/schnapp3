CSG 2D — Solid / Anti-Solid Semantics

Two-dimensional cross-sections illustrating CSG operations. Replace "disk" with "ball" for ℝ³. All diagrams use the two-component (polarity, distance) model.

solid (+1) anti-solid (−1) empty (0)

In each diagram: solid outlines mark the boundary of the named shape; dashed outlines are reference-only (the shape exists but is not the primary actor).

1. The four regions of A and B

Two overlapping solid disks produce four distinct regions.

2. anti(B) — negative charge, same geometry

anti flips polarity but leaves the distance field unchanged. Anti-solid occupies the same region as B but has charge −1.

3. complement(B) — flipped geometry, same charge

complement negates the distance field: inside↔outside swap. The result is an infinite solid with a hole at B. The hole is where B used to be solid.

4. intersect(A, complement(B)) = A − B

intersect keeps only points inside both operands. Inside A and inside complement(B) means: inside A and outside B — the left crescent.

5. Charge propagation: the full 2×2 table

All four combinations of intersect(±A, complement(±B)). Geometry is always A−B (the left crescent). Charge follows the multiplication rule.
